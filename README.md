# Customer (CX) Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior to identify trends, purchasing drivers, and loyalty patterns. The goal is to support data-driven decisions for improving customer engagement and sales strategies.

## Business Problem
How can customer shopping data be leveraged to:
- Identify behavioral trends
- Improve customer engagement
- Optimize marketing and product strategies

## Tools & Technologies
- PostgreSQL (pgAdmin)
- Python (Pandas, NumPy, Matplotlib)
- JupyterLab
- Excel
- Power BI

## Project Workflow
1. Data Cleaning and Transformation using Python
2. Data Structuring and Analysis using SQL
3. Insight Visualization using Power BI
4. Business Recommendations and Reporting

## Key Insights
- Customer demographics impact purchasing behavior
- Discounts and seasonal trends influence repeat purchases
- Online and offline channels show distinct buying patterns
- Payment preferences affect conversion and loyalty
